## Instructions: Complete nanoGPT PR

### Prerequisites
- Fork created: https://github.com/evelynmitchell/nanoGPT ✅

### Steps to Complete PR

```bash
# 1. Clone your nanoGPT fork locally
git clone https://github.com/evelynmitchell/nanoGPT.git
cd nanoGPT

# 2. Create feature branch
git checkout -b add-test-suite

# 3. Create tests directory
mkdir -p tests

# 4. Copy files from Process_Software_Agents
#    (after merging this branch, or from local clone)
cp /path/to/Process_Software_Agents/contrib/nanogpt-tests/__init__.py tests/
cp /path/to/Process_Software_Agents/contrib/nanogpt-tests/conftest.py tests/
cp /path/to/Process_Software_Agents/contrib/nanogpt-tests/test_model.py tests/
cp /path/to/Process_Software_Agents/contrib/nanogpt-tests/test_nanogpt.ipynb tests/
cp /path/to/Process_Software_Agents/contrib/nanogpt-tests/pyproject.toml .

# 5. Verify tests pass
pip install pytest pytest-cov
pytest tests/test_model.py -v

# 6. Commit and push
git add tests/ pyproject.toml
git commit -m "Add test suite for model.py (79% coverage)"
git push origin add-test-suite

# 7. Create PR on GitHub
#    Go to: https://github.com/evelynmitchell/nanoGPT
#    Click "Compare & pull request"
```
